# DIU Event – University Event Management Frontend

This is a frontend-only university project developed as part of a course at
Daffodil International University.

## Project Description
DIU Event is a web-based frontend interface designed to manage university events.
It allows users to view events, explore event details, and understand how a
university event management system could work from a UI perspective.

## Features
- Clean and responsive UI
- Event listing section
- Event details layout
- User-friendly design for students and organizers

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Type
Academic / University Project (Frontend Only)
